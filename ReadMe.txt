Для ввода пользователем массива строк сначала необходимо ввести количество строк.
При вводе осуществляется проверка методом bool Check.
Количество строк необходимо ввести, чтобы зарезервировать необходимый объем памяти для создания массива.
Далее пользователю будет предложено ввести значение каждой строки массива.
Методом int NumberElem определяется количество элементов в заданном массиве, подходящие под условия задачи.
Это также нужно для резервирования памяти для создания результирующего массива.
Методом ArrayThreeSymbChar создается результирующий массив.
Метод PrintArrayHowExample выводит информацию "как в примере".
Программа снабжена выводом на экран промежуточных операций для удобства написания.
Для изменения условия отбора элементов из заданного массива необходимо изменить переменную int symbol (см. строка 77).
